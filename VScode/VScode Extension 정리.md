# VScode Extension

- Material theme
- Live Server
- Prettier

  - Print Width
  - Quote Style
  - Format On Save
  - Format On Type
- Bracket Pair Colorizer (브라켓 괄호 색상)

- CSS Peek (html에서 CSS파일 찾아줌)
- HTML CSS Support(html에서 CSS 자동완성)
